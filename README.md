Назначение программы

Рандомайзер учеников предназначен для случайного выбора ученика из списка. Программа гарантирует, что все ученики будут выбраны прежде, чем кто-то будет выбран повторно.

Основные функции

1. Случайный выбор ученика - нажатие кнопки "Выбрать ученика" запускает процесс случайного выбора
2. История выборов - просмотр последних 10 выбранных учеников с временем выбора
3. Визуальные эффекты - анимация прокрутки номеров и праздничное конфетти при выборе

Как использовать

1. Откройте файл в браузере
2. Нажмите кнопку "Выбрать ученика" для случайного выбора
3. После анимации на экране появится номер и имя выбранного ученика
4. Для просмотра истории нажмите кнопку "История" в правом верхнем углу
5. Чтобы закрыть окно истории, нажмите на крестик или кликните вне окна

Особенности работы

· Программа запоминает, какие ученики уже были выбраны, и не повторяет их, пока не будут выбраны все
· История сохраняет последние 10 выборов с указанием времени
· Визуальные эффекты делают процесс выбора более интересным
